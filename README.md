# Simple Calculator

A basic calculator built with **HTML**, **CSS**, and **JavaScript** that performs simple arithmetic operations: addition, subtraction, multiplication, division and modulo.

## Features
- Clean and responsive interface
- Works on desktop and mobile
- Performs basic calculations

## Files
- `index.html` – Main HTML file
- `style.css` – Styles for the calculator
- `index.js` – JavaScript logic for calculations

## How to Use
1. Open `index.html` in a browser.
2. Click the buttons to perform calculations.
3. Results are displayed instantly.
